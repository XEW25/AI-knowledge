# OpenClaw

## Type
Product / system

## Summary
OpenClaw is an agent system and orchestration environment referenced in [[Alex Zhang - The Mismanaged Geniuses Hypothesis]] as an example of a more general scaffold that lets language models plan, use tools, and coordinate subagents over longer-horizon tasks.

## Relevance to this vault
OpenClaw is relevant both as a referenced agent system and as the environment in which this assistant operates. It is therefore relevant to work on [[Agent orchestration]], [[Task decomposition]], and longer-horizon LM workflows.
